# About this Project

PSDnsClient is a cross-platform PowerShell module for performing DNS lookups and queries, providing a reliable way to query DNS from PowerShell on any Operating System. It aims to get as close as possible to the same output of `Resolve-DnsName` but without the Windows-Only limitations.

## About Dependency DNSClient.Net

This module currently depends on DnsClient.NET version 1.8.0 (NuGet package), an [Apache-2 licensed](https://github.com/MichaCo/DnsClient.NET#Apache-2.0-1-ov-file) C# module.

The DnsClient.DLL and DnsClient.XML are both bundled with this project to ensure availability and assist with packaging. They remain unaltered from the dotNet source.

No NOTICE file was provided by the upstream project.

### From the Github Description

DnsClient.NET is a simple yet very powerful and high performance open source library for the .NET Framework to do DNS lookups.
